# Inference — Test & Live Detection

This folder contains scripts for running inference with the trained YOLO models — both offline testing against extracted frames and live detection using the OAK-D Lite.

---

## Scripts

```
inference/
├── README.md                        ← this file
├── scripts/
│   ├── extract_test_frames.py       ← Step 1: extract test frames from recordings
│   └── test_inference.py            ← Step 2: run inference on all 3 models
├── test_images/                     ← generated by extract_test_frames.py (not in git)
└── inference_results/               ← generated by test_inference.py (not in git)
```

---

## Step 1 — Extract Test Frames (`extract_test_frames.py`)

Extracts **3 evenly-spaced frames** from each recording video and saves them as a lightweight test dataset.

```powershell
python "Object Detection Pipeline\inference\scripts\extract_test_frames.py"
```

**Source:** `DataPrepV3_recordings/cubes/` and `.../cylinders/`
**Output:** `inference/test_images/cubes/` and `.../cylinders/`

The frames are taken from the start, middle, and end of each video (avoiding the very first and last frame) to get representative coverage. With 12 input videos (6 cubes, 6 cylinders) this produces **36 test images**.

> The `test_images/` folder is not committed to git — regenerate it locally by running this script.

---

## Step 2 — Run Test Inference (`test_inference.py`)

Runs all three trained YOLO models against the test images, saves annotated results, and prints a detection summary table.

```powershell
# Run all three models
python "Object Detection Pipeline\inference\scripts\test_inference.py"

# Run a single model
python "Object Detection Pipeline\inference\scripts\test_inference.py" --model v8
python "Object Detection Pipeline\inference\scripts\test_inference.py" --model v26
python "Object Detection Pipeline\inference\scripts\test_inference.py" --model v5

# Override the confidence threshold (applies to all models)
python "Object Detection Pipeline\inference\scripts\test_inference.py" --conf 0.4

# Single model with custom threshold
python "Object Detection Pipeline\inference\scripts\test_inference.py" --model v5 --conf 0.3
```

**Output:** Annotated images saved to `inference/inference_results/<model_name>/`

> The `inference_results/` folder is not committed to git.

### Default Confidence Thresholds

Each model uses its own optimal threshold derived from the F1-Confidence curve analysis:

| Model | Default Threshold | Source |
|---|---|---|
| YOLOv8n | 0.566 | Peak F1 = 0.93 at 0.566 |
| YOLO26n | 0.565 | Peak F1 = 0.92 at 0.565 |
| YOLOv5n | 0.405 | Peak F1 = 0.93 at 0.405 |

> **Note:** Thresholds for v8n/v26n are not interchangeable with v5n — anchor-based and anchor-free models calibrate confidence scores differently.

## Troubleshooting

| Error | Fix |
|---|---|
| `No images found in test_images/` | Run `extract_test_frames.py` first |
| `weights not found` | Check `WEIGHTS_DIR` path in `test_inference.py` |
| `ModuleNotFoundError: ultralytics` | `pip install ultralytics` |
| `Could not open <video>` | Video file missing or corrupt — check `DataPrepV3_recordings/` |
